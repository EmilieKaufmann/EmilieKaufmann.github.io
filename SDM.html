  • Lecture 2: Stochastic Multi-Armed Bandits.

  • Lecture 4: Markov Decision Processes. Notebook.
  • Lecture 5: Dynamic Programming. Notebook.
  • Lecture 6: Reinforcement Learning Algorithms.
  • Lecture 7: Reinforcement Learning with Function Approximation.
  • Practical session 2: RL with and without function approximation. Link to do it with Google collab.
  • Lecture 8: Beyond Value-Based Algorithms.
